# Garatujas

## HTML para eu do futuro

Esse repositorio é um bloco de anotacoes sobre tudo que teremos ao longo do ano

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para estruturar e organizar páginas na web. Ela define o esqueleto de um site, como títulos, parágrafos e imagens, permitindo que navegadores (Chrome, Firefox) saibam como exibir o conteúdo. Não é uma linguagem de programação, mas sim de estrutura.

CSS (Cascading Style Sheets, ou Folhas de Estilo em Cascata) é uma linguagem de estilização usada para definir a aparência visual de páginas web criadas com HTML. Enquanto o HTML estrutura o conteúdo, o CSS controla layout, cores, fontes, espaçamentos e responsividade, separando a estrutura da apresentação para tornar sites atraentes e organizados.
