# Garatujas

## HTML para eu do futuro

Esse repositorio é um bloco de anotacoes sobre tudo que teremos ao longo do ano

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental utilizada para estruturar e criar páginas na internet. Ela funciona como o "esqueleto" de um site, organizando textos, imagens, vídeos e links para que o navegador (como Chrome ou Firefox) saiba como exibir o conteúdo.

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). É uma linguagem de estilo utilizada para definir a aparência visual de documentos escritos em HTML ou XHTML.
